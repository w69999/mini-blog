# Mini Blog / CMS ✨

## Opis projektu
Projekt przedstawia prosty mini CMS (Content Management System) w formie bloga,
stworzony z wykorzystaniem nowoczesnych technologii webowych.
Aplikacja działa po stronie klienta (client-side).

## Funkcjonalności
- Dodawanie, edycja i usuwanie postów (CRUD)
- Wyświetlanie listy postów oraz pojedynczego posta
- Responsywny interfejs użytkownika
- Dynamiczna manipulacja DOM
- Przechowywanie danych w localStorage
- Generowanie treści posta z wykorzystaniem AI (mock API)

## Technologie
- HTML5
- CSS3 (Flexbox)
- JavaScript (ES6)
- Web Storage API (localStorage)
- Git & GitHub


## Uruchomienie projektu
1. Sklonuj repozytorium
2. Otwórz plik `index.html` w przeglądarce

## Ograniczenia
Projekt nie wykorzystuje backendu ani rzeczywistej bazy danych.
Dane są przechowywane lokalnie w przeglądarce użytkownika.
